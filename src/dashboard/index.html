Framework & Metodologi

Arsitektur sistem SmartBudget Lakehouse untuk deteksi anomali dan transparansi APBD Kota Surabaya.

layers Medallion Architecture (Lakehouse)

B

Bronze Layer

Data mentah dari API CKAN Open Data Surabaya. Disimpan tanpa transformasi sebagai sumber kebenaran tunggal.

Format: CSV • Sumber: CKAN API
S

Silver Layer

Data yang sudah dibersihkan, dinormalisasi, dan diperkaya dengan fitur turunan (rasio, z-score, log transform).

Proses: Pandas • Output: silver_apbd_belanja.csv
G

Gold Layer

Data siap konsumsi dengan skor anomali dari Machine Learning. Langsung disajikan ke dashboard.

Model: Isolation Forest • Output: apbd_scored.csv

psychology Pipeline Machine Learning

1

Feature Engineering

Ekstraksi 11 fitur numerik: log_realisasi, log_pagu, rasio_realisasi_pagu, zscore_per_skpd, trailing_zeros, is_q4, is_december, vendor_kontrak_count, dll.

2

Isolation Forest (Unsupervised)

Algoritma deteksi outlier berbasis partisi acak. n_estimators=200, contamination=0.05. Mampu mendeteksi pola pengeluaran janggal tanpa data label.

3

Scoring & Classification

Setiap transaksi diberi anomaly_score (0-1). Score > threshold diklasifikasikan sebagai "Anomali" untuk investigasi lebih lanjut oleh auditor.
